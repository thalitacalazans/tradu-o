Uso de Gráficos de Flot

Flot é uma biblioteca de plotagem JavaScript pura para jQuery, com foco no uso simples, aparência atraente e recursos interativos. No SB Admin, estamos usando a versão mais recente do Flot juntamente com alguns plugins para melhorar a experiência do usuário. Os plugins de Flot sendo usados ​​são o plug-in de dica de ferramenta para dicas de ferramentas flutuantes e o plug-in de redimensionamento para gráficos totalmente responsivos. A documentação para Flot Charts está disponível em seu site, http://www.flotcharts.org/.
