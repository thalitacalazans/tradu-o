Informações de Uso de Tabelas de Dados

DataTables é um plugin de tabelas muito flexível e avançado para jQuery. No SB Admin, estamos usando uma versão especializada do DataTables criado para o Bootstrap 3. Também personalizamos os títulos das tabelas para usar os ícones do Font Awesome no lugar das imagens. Para obter a documentação completa do DataTables, visite o website deles em https://datatables.net/.
